# Book Sales Data Analysis Project

## Overview
This project focuses on analyzing a book sales dataset to uncover trends related to sales performance, pricing, genres, languages, authors, and publishers. The goal was to clean raw data, handle missing values, and generate insights through exploratory data analysis and visualization.

## Dataset
The dataset contains information on book titles, authors, publishers, genres, languages, sales figures, ratings, and pricing. It required extensive cleaning before analysis, including column standardization and handling missing or inconsistent values.

## Data Cleaning & Preparation
- Standardized column names (lowercase, removed spaces and special characters)
- Converted publishing year to numeric format
- Handled missing values using mean and mode imputation
- Removed numeric noise from book titles
- Verified data types and structure

## Key Analysis Performed
- Sales performance by author and publisher
- Relationship between sale price and units sold
- Distribution of books by language
- Genre comparison using boxplots
- Ratings and review count analysis
- Revenue contribution by publishers

## Key Insights
- Certain publishers dominate total revenue across the dataset
- Book price shows a noticeable relationship with units sold
- Specific genres receive significantly higher engagement and ratings
- English-language books make up the majority of sales
- Author reputation (ratings) influences reader engagement

## Visualizations
- Boxplots for genre vs. ratings count
- Scatter plots for price vs. units sold
- Pie chart showing language distribution
- Bar charts for author and publisher performance

## Output
- Cleaned dataset exported for further analysis
- Visual insights to support data-driven decision-making
